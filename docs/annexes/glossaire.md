# Glossaire

## Termes et définitions

### A

**API (Application Programming Interface)**
Interface de programmation permettant à des logiciels de communiquer entre eux.

### H

**Hyperviseur**
Logiciel permettant de créer et gérer des machines virtuelles. Il existe deux types :
- **Type 1 (bare-metal)** : S'exécute directement sur le matériel (ex: ESXi, Hyper-V, Proxmox VE, XCP-ng)
- **Type 2 (hosted)** : S'exécute sur un système d'exploitation hôte (ex: VMware Workstation, VirtualBox)

### N

**Nested Virtualization (Virtualisation imbriquée)**
Technique permettant d'exécuter un hyperviseur à l'intérieur d'une machine virtuelle.

### V

**VM (Virtual Machine)**
Machine virtuelle émulant un ordinateur complet avec son propre système d'exploitation.

**VLAN (Virtual Local Area Network)**
Réseau local virtuel permettant de segmenter logiquement un réseau physique.

**vSwitch (Virtual Switch)**
Commutateur réseau virtuel permettant de connecter des machines virtuelles entre elles et avec le réseau physique.
